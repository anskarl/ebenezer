Hive

case class Hive[A](action: (HiveConf, IMetaStoreClient) ⇒ Result[A]) extends Product with Serializable

A data-type that represents a Hive operation.

Hive operations use a HiveConf as context, and produce a (potentially failing) result. For convenience Hive operations receive both a HiveConf and a handle to a Hive client. The client is created from the HiveConf when the run method is called.
